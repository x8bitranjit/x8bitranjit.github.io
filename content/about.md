### What's inside each kit
- **Testing Guide**: find / bypass / impact / validity / severity / reporting, with "if blocked, do X next" playbooks.
- **Attack Arsenal**: copy-paste payloads & commands.
- **Checklist**: tick-per-target testing order.
- **PoC Scripts**: runnable, benign-by-default proof-of-concept code.
- **Zero to Expert (Q&A)**: fundamentals / advanced, with real-world CVEs & references.

### Focus
Impact-first, zero-false-positive, modern, stealthy. Authorized testing only: benign markers, own test
accounts, clean up artifacts. **Test only systems you have written permission to test** (in-scope bug
bounty, signed engagements, CTFs, your own labs).
